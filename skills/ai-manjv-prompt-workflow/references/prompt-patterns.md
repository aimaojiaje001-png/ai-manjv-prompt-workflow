# AI 漫剧生图与短剧提示词模板

按任务选择模板。只保留生图、资产图、故事板、首尾帧、短视频镜头、连续性和表演相关内容。不要收录平台邀请码、PSD 工具、纯配音网站或纯编剧大师设定。

## 工作流总览

```text
输入【小说原文/剧情梗概/镜头想法/参考图】后，按顺序输出：
1. 剧情压缩：按时间线提炼动作、冲突、结果和人物反应。
2. 资产清单：角色、场景、道具、色板、关键状态。
3. 生图提示词：角色设定图、场景资产图、道具特写、色卡。
4. 分镜提示词：故事板制作板或线稿分镜表。
5. 视频提示词：首尾帧、一镜到底、连续动作、表演细节。
6. 连续性检查：相邻镜头、段落衔接、剪辑动机和负面限制。
```

## 剧情压缩

用于把小说章节转成可分镜的短剧动作线。

```text
以剧情推进为主线，按原文顺序拆成短句。每句必须包含：
- 谁做了什么或发生了什么；
- 造成的关键结果、人物反应或关系变化；
- 与画面有关的地点、动作、道具或情绪。

不要单独罗列外貌和背景。视觉信息要融入动作、冲突、结果或人物反应。
若目标是 15 秒视频，进一步拆成 0-3s、3-6s、6-10s、10-13s、13-15s 的动作节奏。
```

## 角色视觉 DNA

用于从小说里生成角色资产，而不是模板化帅哥美女。

```text
身份：影视角色设计师、概念艺术总监、电影美术指导。
目标：从原文建立角色视觉 DNA，并输出可用于角色设定图、三视图、表情和状态资产的生图提示词。

提取：
【身份】【年龄段】【性别】【社会阶层】【成长环境】【当前处境】【核心欲望】【最大恐惧】【关键经历】【性格优点】【性格缺陷】【行为习惯】

生成：
【视觉符号】必须有来源故事；
【发型】来自职业、环境、生活状态；
【服装状态】体现身份、经济条件、战损或剧情阶段；
【配饰/道具】体现信仰、阶层、经历或关系；
【身体特征】伤痕、肤色、体态、训练痕迹、职业痕迹。

限制：
禁止网红脸、偶像脸、所有角色长一样、空洞美貌词。所有视觉特征必须服务人物经历。
```

## 万物生式人物与场景资产拆解

用于用户提到“6.13”“万物生”“高质量人物和场景资产图”“沙漠村庄场景”时，从故事里建立可复用资产库。

```text
身份：影视前期视觉开发总监、角色设计师、场景概念设计师、AI 短剧资产制片。

输出：
1. 故事视觉母题：时代/地域、冲突来源、主色调、材质、镜头气质。
2. 人物资产：主角、配角、反派、群众或非人角色；每个角色列身份、剧情功能、当前状态、视觉 DNA、服装、体态、伤痕/职业痕迹、标志性道具、可复用表情。
3. 状态资产：同一角色拆【常态】【战损/疲惫】【爆发/觉醒】【伪装/隐藏】等阶段。
4. 场景资产：外部大环境、核心行动空间、过渡空间、关键局部、可复用背景。
5. 道具资产：推动剧情或强化身份的物件，写来源、材质、磨损、尺寸、颜色、与角色关系。
6. 优先级：S 级先生成角色一致性和主场景，A 级支撑多镜头复用，B 级补氛围。

对沙漠村庄类场景，至少拆：远景村落轮廓、村口道路、民居材质、公共空间、室内生活细节、风沙光照、地貌动线。
```

## 人物多角度设定图

```text
生成一张角色三视图 / character turnaround sheet。

角色：【整合视觉 DNA 后的人物描述】
内容：正面、侧面、背面、3/4 角度、半身特写、表情特写、放松站姿、服装细节、配饰细节、配色方案。
一致性：同一张脸、同一套服装、同一发型、同一体型、同一配饰；不要多人、换脸、换装、比例漂移。
排版：浅色干净背景，网格布局，角色完整展示，不裁切。
画质：高清、真实材质、清楚边缘、柔和布光、可读标注。
负面：乱码、伪文字、随机字母、水印、模糊、脏乱纹理、过度锐化。
```

## 场景资产图

```text
生成一张可供短剧多镜头复用的场景资产图。

场景：【时代/地点/空间层级/剧情用途/核心氛围】
主画面：全景或俯瞰图，说明空间秩序和人物可移动区域。
细节图：关键建筑轮廓、纵深路径、地面材质、门窗器物、灯光来源、可互动道具。
风格：【电影写实/古风/玄幻/科幻/校园/悬疑等】
光影：【时间、色温、雾气、空气透视、装饰光】
构图：左侧主场景，右侧多格细节，必要时加简单序号。
避免：时代错乱、现代违和物、建筑变形、透视错误、杂乱、过度文字、水印。
```

## 道具与局部资产

```text
为【道具/配饰/关键局部】生成资产参考图。
必须说明：剧情功能、持有人、尺寸、材质、磨损、颜色、可读标识、与角色/事件关系。
输出形式：正面图、侧面图、局部特写、使用状态、与手/身体/空间的比例关系。
限制：不要变成完整人物海报；不要让背景抢主物；关键文字建议后期叠字。
```

## 色卡参考图

```text
根据参考图生成 16:9 扁平化数字色卡，不是照片、插画或实物摆拍。

包含：
- 中性浅色纯背景；
- 顶部标题和副标题，像电影美术设定页；
- 10-15 个横向色块，默认 13 个；
- 每个色块给 HEX 和功能标签；
- 色块顺序：环境主色/光照色 -> 中间调/阴影色 -> 材质色 -> 强调色。

避免：摄影渲染、纸张纹理、桌面、阴影、景深、胶片颗粒、3D、手绘、水彩、mockup。
```

## 故事板 2.0

用于 Image2 或类似图像模型生成导演预制作板。

```text
输入只写“发生了什么”，先简化剧情，再生成资产，最后生成故事板。

故事 =【连续动作和情绪变化，不要只写概念】

输出一个电影制作板/视觉规划表：
1. 共享创意指导：镜头数量、调色、环境、画幅。
2. 角色参考：同一角色多角度、服装、配饰、状态。
3. 场景设计：空间结构、移动路径、摄像机位置。
4. 编号故事板：约 6-8 帧；每帧写景别、镜头运动、动作、情绪、时长。
5. 灯光/色彩/气氛：明确光源、时间、质感。
6. 音频/节奏：环境声、动作声、音乐或无配乐。

避免场景过于相似；每帧都必须承担动作、信息、情绪或空间功能。
```

## 奇幻动作故事板

用于“红衣少女+神兽”“法天相地”“白衣女剑仙”等强动作/奇观短片。

```text
故事 =【角色状态 -> 触发动作 -> 奇观显现 -> 运动推进 -> 情绪落点】

要求：
- 角色、神兽/法相/武器必须有清晰比例关系；
- 动作要连续，不能每格都像独立海报；
- 用体积光、尘土、落叶、能量流、衣摆等元素表现速度和方向；
- 故事板下方可附主场景图、俯视图、移动路线和光影色板；
- 标注每镜头的主体、动作、景别、时长和转场意图。

画幅按任务选择：短剧生产默认 16:9；若用户明确要求竖屏/3:4，保留用户画幅。
```

## 线稿分镜拆解

用于把小说转成可生图、可生视频的镜头表。

```text
身份：影视分镜拆解师。
任务：不是概括剧情，也不是逐句翻译，而是拆成连续影视镜头。

每个镜头包含：
【场次】【镜号】【画面草图】【景别】【机位/角度】【镜头运动】【动作/调度】【台词/声音】【时长】【涉及素材】【备注】【生成提示词】

关键规则：
- 画面字段写观众看见什么，保留具体空间层级。
- 结果/备注字段写叙事功能、关系变化、压迫递进、信息揭露或身体后果。
- 文戏必须有潜台词或关系变化；动作戏必须有明确身体/空间结果。
- 线稿只是保障，核心是分镜词和连续性。
```

## 短剧镜头提示词

用于从分镜表转成 Seedance/SD2.0/类似视频模型提示词。

```text
镜号：【编号】
风格：【真人电影/院线电影/写实短剧/古风/玄幻/科幻】
空间：【场景资产名；说明相邻空间不要混成一个空间】
景别：【远景/全景/中景/近景/特写】
机位/角度：【平视/俯视/仰视/过肩/主观等】
镜头运动：【固定/推进/跟拍/环绕/手持等】
动作/调度：【人物怎么动、道具怎么动、空间关系怎么变】
声音/台词：【可选】
时长：【秒数】
涉及素材：【角色、场景、道具】
生成提示词：【一句可直接给视频模型的镜头描述】
负面：【跳切、换脸、换装、空间错乱、多余人物、水印、字幕】
```

## 连续动作视频

```text
把约 15 秒动作拆成：
0-3s：起幅和初始状态；
3-6s：动作启动；
6-10s：动作主体和速度变化；
10-13s：动作收束或空间变化；
13-15s：定格、转场或情绪落点。

写清身体相位、重心、视线、手脚、服装/道具、镜头运动和情绪递进。
不要只写“激烈”“炫酷”“好看”；要写能被模型连续执行的动作。
```

## 首尾帧与一镜到底

```text
首帧生图：
16:9 横版电影剧照；不要分镜格、四宫格、边框、字幕或 logo。
写清地点、时间、光色、空间结构、人物年龄/身份/服装/姿态/心理状态、构图层次和摄影风格。

尾帧生图：
延续同一空间、同一人物、同一服装、同一主道具和光线方向，只改变动作结果与情绪落点。

视频提示词：
生成一条连续长镜头，说明摄像机如何移动、景别如何变化、人物如何从首帧自然过渡到尾帧。

负面：
no grid, no storyboard layout, no captions, no subtitles, no logo, no split screen, no collage, no multiple frames, no extra people, no distorted hands, no jump cut.
```

## 机甲变身首尾帧连续

用于首尾帧无缝衔接的变身短片。

```text
资产：
1. 人类角色多角度设定图；
2. 机甲形态多角度设定图；
3. 场景资产图；
4. 首帧和尾帧。

视频节奏：
0-2s：人类状态进入画面，镜头锁定主体；
2-4s：能量或材质变化触发；
4-10s：装甲/结构逐层生成，变身过程占主要时长；
10-13s：头部、推进器、武器或关键结构完成；
13-15s：变身完成并进入稳定姿态。

约束：
主体始终占画面主导；背景不抢戏；人类和机甲身份连续；不要突然换脸、跳切、缺失肢体或凭空出现完整机甲。
衔接下一段时，用上一段尾帧作为下一段首帧，并重新挂接角色/机甲/场景参考。
```

## 活人感表演

用于视频提示词或让思考模型细化表演。

```text
先判断人物心理动机：解释、隐忍、回忆、撒谎、后悔、试探、故作轻松等。
再设计眼神、表情、停顿、语速、嘴角、眉眼、肩颈、手指、身体重心。

正面写法：
【人物】不是直接表现【情绪】，而是在努力【控制/掩饰/试探/回忆】。说话前【微动作】，抬眼时【视线变化】，关键处【停顿/语速变化】，最后出现【细微表情变化】。

负面限制：
固定假笑、眼神空洞、表情突然切换、夸张哭笑、脸部情绪和台词含义不一致、语气和表情脱节、一直盯镜头、没有思考感、像在念台词。
```

## 分镜连续性检查

```text
C-to-C：检查相邻镜头的动作相位、180 度轴线、屏幕方向、30 度规则、景别差异、构图相似、视线匹配、运动向量、服装/伤痕/道具/环境/光线/天气、声音延续和剪辑动机。

S-to-S：从第二段开始，比较上一段结尾和当前段 C01 开头；重查位置、方向、视线、道具、衣伤、环境、光线、天气、声音尾巴和情绪能量。

剪辑动机至少有一个：
动作匹配、视线揭示、反应、信息/物件、节奏冲击、空间重建、声音桥、时间/场景转换。

拒绝：
无动机硬切、相似构图堆叠、可删除且不损失动作/情绪/信息/连续性的镜头块。故意跳切必须事先声明目的和节奏规则。
```
